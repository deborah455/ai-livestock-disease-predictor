# AI Livestock Disease Predictor Using Vision, Sensors & Fusion Modeling

## Overview

An AI-powered livestock health monitoring system that combines computer vision and sensor analytics to detect potential animal diseases.

The project integrates image-based disease detection using deep learning with physiological sensor analysis using machine learning. Predictions from both models are combined through a fusion algorithm to improve diagnostic reliability.

## Features

* Disease detection from livestock images using MobileNetV2
* Sensor-based health risk prediction using Random Forest
* Multi-modal fusion modeling
* Real-time inference interface
* Prediction logging and analysis
* ROC curve and confusion matrix evaluation
* Feature importance visualization

## System Architecture

1. Image preprocessing and augmentation
2. MobileNetV2 vision model training
3. Sensor data simulation and preprocessing
4. Random Forest sensor model training
5. Fusion of vision and sensor probabilities
6. Real-time prediction and evaluation

## Results

The fusion model demonstrated improved prediction performance compared to individual vision-only and sensor-only approaches.

Evaluation included:

* Accuracy
* F1 Score
* AUROC
* Confusion Matrices
* ROC Curves
